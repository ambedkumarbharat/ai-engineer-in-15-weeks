## 🎯 Why This Phase?

RAG is the **#1 most in-demand AI engineering skill**. It's how companies add AI to their products without fine-tuning models. You'll learn to build systems that retrieve relevant context from a knowledge base and use LLMs to generate accurate, cited answers.

## ✅ Phase Completion Criteria

- [ ] Generate and store embeddings using sentence-transformers or OpenAI
- [ ] Build a complete RAG pipeline: load → chunk → embed → store → retrieve → generate
- [ ] Evaluate RAG quality with faithfulness and relevance metrics
- [ ] Implement hybrid search (vector + keyword) with result fusion
