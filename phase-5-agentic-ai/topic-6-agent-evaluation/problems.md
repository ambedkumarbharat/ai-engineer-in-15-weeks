## Problem 01 🧩 **🟢 Easy** — **Task Completion Tracker**
Build a system that tracks whether an agent successfully completes simple tasks (return correct answer, use right tool, stay within budget).

## Problem 02 🧩 **🟢 Easy** — **Step Counter**
Build an evaluation wrapper that counts tool calls, LLM calls, total tokens, and execution time for every agent run.

## Problem 03 🧩 **🟡 Medium** — **Trajectory Evaluator**
Build a system that evaluates whether the agent took the optimal path (fewest steps, correct tools, no unnecessary actions).

## Problem 04 🧩 **🟡 Medium** — **Cost Tracker**
Build a comprehensive cost tracker: per-step cost, per-tool cost, per-model cost, total cost. Generate cost reports.

## Problem 05 🧩 **🟡 Medium** — **Safety Evaluator**
Build a safety evaluation framework: check if the agent attempted harmful actions, leaked data, or exceeded permissions.

## Problem 06 🧩 **🟡 Medium** — **Regression Test Suite**
Build 20 test cases for your agent. Run after every code change. Alert if success rate drops or cost increases.

## Problem 07 🧩 **🔴 Hard** — **LLM-as-Judge for Agents**
Use an LLM to evaluate agent outputs on quality, completeness, and efficiency. Compare with human evaluations.

## Problem 08 🧩 **🔴 Hard** — **Multi-Dimension Scoring**
Build a scoring system: task_completion(0-1) × accuracy(0-1) × efficiency(0-1) × cost_score(0-1). Rank agent configurations.

## Problem 09 🧩 **🔴 Hard** — **A/B Testing Framework**
Build a framework that A/B tests two agent configurations on 50 tasks, measures all metrics, and determines statistical significance.

## Problem 10 🧩 **🔴 Hard** — **Agent Leaderboard**
Build a leaderboard that benchmarks multiple agent implementations (ReAct, Plan-Execute, LangGraph) on the same task suite.
