## Problem 01 🧩 **🟢 Easy** — **Two-Agent Crew**
Build a simple 2-agent crew: researcher finds information, writer creates a summary. Use sequential process.

## Problem 02 🧩 **🟢 Easy** — **Agent Role Design**
Design 5 specialized agent personas with detailed roles, goals, and backstories for: data analyst, security auditor, UX designer, DevOps engineer, and ML engineer.

## Problem 03 🧩 **🟡 Medium** — **Research Report Crew**
Build a 3-agent crew (researcher → analyst → writer) that produces a structured research report on any given topic.

## Problem 04 🧩 **🟡 Medium** — **Code Review Crew**
Build a crew with a coder who writes code and a reviewer who critiques it. The coder improves based on feedback in iterative rounds.

## Problem 05 🧩 **🟡 Medium** — **Task Dependencies**
Build a crew with 5 tasks where task 3 depends on tasks 1 and 2, and task 5 depends on tasks 3 and 4. Use context passing.

## Problem 06 🧩 **🟡 Medium** — **Agent with Custom Tools**
Give each agent in a crew specialized tools: researcher gets web search, data analyst gets calculator + chart generator, writer gets file writer.

## Problem 07 🧩 **🔴 Hard** — **Hierarchical Crew**
Build a hierarchical crew with a manager agent that delegates to specialized sub-agents and merges their work.

## Problem 08 🧩 **🔴 Hard** — **Competitive Agents**
Build two agents that independently solve the same problem, then a judge agent picks the better solution. Compare with single-agent accuracy.

## Problem 09 🧩 **🔴 Hard** — **Multi-Crew Pipeline**
Chain multiple crews together: Crew 1 (research) → Crew 2 (analysis) → Crew 3 (reporting). Pass output between crews.

## Problem 10 🧩 **🔴 Hard** — **Crew Cost Optimizer**
Build a system that tracks token usage per agent in a crew, identifies which agents are expensive, and suggests optimizations (shorter prompts, cheaper models).
