## Problem 01 🧩
**Difficulty:** 🟢 Easy
### System Prompt Designer
**Problem:** Design 5 different system prompts for: code reviewer, SQL tutor, technical writer, data analyst, and DevOps expert. Each should define role, rules, and output format.

## Problem 02 🧩
**Difficulty:** 🟢 Easy
### Few-Shot Text Classifier
**Problem:** Build a text classifier using few-shot prompting with 3 examples per category. Classify support tickets into: billing, technical, feature_request, bug_report.

## Problem 03 🧩
**Difficulty:** 🟡 Medium
### Chain-of-Thought Problem Solver
**Problem:** Create a prompt that solves multi-step word problems using chain-of-thought reasoning. Compare accuracy with and without CoT on 20 test problems.

## Problem 04 🧩
**Difficulty:** 🟡 Medium
### JSON Output Enforcer
**Problem:** Design prompts that reliably produce valid JSON output. Handle edge cases: nested objects, arrays, special characters, null values. Build a validation layer.

## Problem 05 🧩
**Difficulty:** 🟡 Medium
### Prompt Template Library
**Problem:** Build a reusable prompt template library with: RAG QA, summarization, classification, extraction, and comparison templates. Support variable substitution and chaining.

## Problem 06 🧩
**Difficulty:** 🟡 Medium
### Prompt Injection Tester
**Problem:** Create a test suite of 20 prompt injection attacks and design system prompts that resist them. Measure defense success rate.

## Problem 07 🧩
**Difficulty:** 🟡 Medium
### Multi-Language Prompt Adapter
**Problem:** Build prompts that work consistently across English, Hindi, and Spanish inputs. Measure output quality in each language.

## Problem 08 🧩
**Difficulty:** 🔴 Hard
### Prompt A/B Testing Framework
**Problem:** Build a framework that A/B tests two prompt variants on 50 test cases, measures quality metrics, and determines statistical significance (p-value < 0.05).

## Problem 09 🧩
**Difficulty:** 🔴 Hard
### Self-Improving Prompt Generator
**Problem:** Build a system where an LLM generates prompts, evaluates their outputs, and iteratively improves the prompts based on evaluation scores.

## Problem 10 🧩
**Difficulty:** 🔴 Hard
### Role-Play Consistency Checker
**Problem:** Create a system where an LLM maintains a character persona across 50+ conversation turns without breaking character. Measure consistency.
